Tools and Data to Replicate our Experiments

Below are all repositories linked to (i) reproduce the results of our paper "Scanning the IPv6 Internet Using Subnet-Router Anycast Probing" and (ii) run your own SRA measurements.

Paper Artifacts

Repository: netd-tud/artifacts-ipv6-sra-scanning

This repository contains the artifacts to reproduce plots and tables, and includes the tools to run your own IPv6 SRA measurements.

Measurement Data (OPARA)

Dataset: doi.org/10.25532/OPARA-979

This dataset is part of the artifact collection and contains all the necessary dataframes to reproduce paper and tables of our paper.

IPv6 Address Generator

Repository: mkoch-tud/go_ipv6_address_generator

Fast Go-based address generator. Pass an input file with IPv6 prefixes and a target network size, the tool generates all possible subnets between (and including) the initial prefix and the target prefix size.

ZMap SRA Extensions

Repository: mkoch-tud/zmap

Customized ZMap fork from TU Munich with an ICMPv6 router discovery module for Subnet-Router anycast probing -- use together with the GO address generator.
